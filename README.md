# COVID-19-Early-Case-Trend-Analysis-Recovery-Insights
COVID-19 trend &amp; recovery analysis using Python and ML

# Overview
This project analyases the data of early covid-19 patients to identify patterns,demographic trends and recovery insights

# Tech Stack
Python
Pandas
Matplotlib
Seaborn
Scikit-learn

# Features
Performs Exploratory Data Analysis (EDA)
Data Visualization
Recovery Time Analysis
Linear Regression Model for prediction

# How to run ?
python main.py

# Output
Age distribution
Recovery trends
Correlation heatmap
ML model performance (R² score)
